vim 常用命令

发表于 2016-08-11 | 分类于 Linux

一般模式

  • 在当前行下一行开始编辑 o

  • 在当前行上一行开始编辑 O

  • 删除单个字符 x

  • 删除/剪切当前行 dd

  • 复制当前行 yy

  • 对复制/剪切的行进行粘贴 p

  • 撤销 u

  • 恢复 Ctrl + R

  • 跳到指定行 <行号>gg

  • 跳到文首 gg

  • 跳到文尾 GG

  • 视图模式 v

命令模式

  • 设置行号 :set nu

  • 打开高亮 :set hls

  • 关闭高亮 :set nohls

  • 查找 :/<string>

    • 向下查找 n
    • 向上查找 N

  • 替换 %s/<old_str>/<new_str>/g

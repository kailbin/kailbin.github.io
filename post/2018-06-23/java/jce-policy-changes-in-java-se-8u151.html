发表于 2018-06-23 | 分类于 Java |

随着越来越多的第三方工具只支持 JDK8，最近公司也计划从 JDK7 升级到 JDK8，在线下环境升级过程中，发现某些项目报以下异常：

这是因为某些国家的进口管制限制，JDK默认的加解密有一定的限制。
比如默认不允许 256 位密钥的 AES 加解密，解决方法就下载官方JCE无限制强度加密策略文件，覆盖即可。

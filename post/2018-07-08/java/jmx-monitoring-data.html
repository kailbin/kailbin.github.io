通过 JMX 获取 JVM 监控数据

发表于 2018-07-08 | 更新于 2018-07-15 | 分类于 Java | 评论数：

本文 提供了一个 Java 工具类，用来获取诸如以下的 JVM 数据：

  • 类加载信息
  • 编译信息
  • 操作系统信息
    • 系统名称、架构、版本、负载 …
    • 物理内存信息
    • CPU 信息
    • 等
  • 运行时信息
    • 进程ID
    • JVM 版本信息
    • 启动参数
    • 系统属性
    • 启动时间、运行时常
    • 等
  • 线程信息
    • 线程状态
    • 死锁检查
    • 等
  • JVM 内存信息
    • 内存使用情况
    • 各个区域内存使用情况
    • 垃圾收集器信息
    • 等
